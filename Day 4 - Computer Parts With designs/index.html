Information of Computer Parts:

Central Processing Unit (CPU). Processor or brain of the computer. Performs all calculations and tasks, processes instructions from programs.
The main circuit board connecting all components (CPU, RAM, storage, GPU, etc.).
Random Access Memory (RAM). Temporary memory that stores data currently being used by programs.
Hard Disk Drive (HDD). Stores the operating system, programs, and files.
Solid State Drive (SSD). Stores the operating system, programs, and files.
Power Supply Unit (PSU). Converts electricity from the outlet to usable power for the computer components.
Graphics Processing Unit (GPU). Renders images, videos, and 3D graphics.
CPU Coolear Fan. Keeps components, especially the CPU and GPU, from overheating.
Computer Case (Chassis). Houses and protects all internal components.
Network Interface Card (NIC).network adapter or Ethernet card, is a hardware component that allows a computer to connect to a network.
Wi-Fi Card Adapter (if not built-in). is used to enable wireless internet access for devices that don't have built-in Wi-Fi or need an upgrade.
CMOS Battery. maintains power to the BIOS settings and the real-time clock even when the computer is off.
Monitor. primarily used as an output device to display visual information from a computer.
Keyboard. used to enter characters and functions into a computer system.
Mouse. used to interact with computers by controlling the cursor on the screen.
Speakers. used to output audio, providing sound effects, music, voice-overs, and other audio elements.
Microphone. recording music and voices to communication in telephones and radio.
Webcam. video communication and recording
USB Port. connecting a wide variety of devices to computers, transferring data, and providing power.
